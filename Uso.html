Afinal, onde o Javascript pode ser usado?

O Javascript é a linguagem mais utilizada para programação de aplicações Web
e é suportada pela maioria dos navegadores (Firefox, Chrome, Edge e Safari)
sem que o usuário precise instalar um plug-in.

Hoje em dia pode ser utilizada no desenvolvimento do Back-End junto com o Node.JS,
um software de código aberto e multiplataforma e o framework Express,
ampliando o campo de atuação de um desenvolvedor para Full-Stack.
Também é utilizado para aplicações em mobiles com uma infinidade de frameworks e no desenvolvimento de jogos!
